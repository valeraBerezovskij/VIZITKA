3D molecular visualizer

Description

"3D Molecule Visualizer" is an interactive application designed to visual studying molecular structure. The project allows you to create and explore three-dimensional models of molecules, What simplifies the understanding of complex chemical processes.

The application supports downloading molecular data in various formats (eg PDB, XYZ), provides tools for manipulating molecules (rotating, scaling, selecting atoms) and offers various types of visualization (ball-and-stick, solid, space-filling) models.

Key features of the project:

  • High performance
  • Intuitive interface
  • Supports various data formats
  • Wide range of visualization options

Technologies

  • C++
  • Qt
  • OpenGL
  • Eigen
  • Assimp
